Práctica de Despliegue en Azure

Sobre el proyecto

Esta página web es el resultado de una práctica técnica de despliegue en la nube. Actualmente está alojada en una Máquina Virtual de Microsoft Azure. El contenido es servido a través del servidor web Nginx y la conexión está asegurada mediante certificados SSL.

Tecnologías

Azure
Ubuntu
Nginx
